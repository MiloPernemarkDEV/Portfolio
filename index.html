NATIVE PLUGIN / ENGINE UTILITY
C++ / C# / Unity Native Plugins (P/Invoke)

Native Vector Math & Pong Clone

Engineered a high-performance native C++ linear algebra library and seamlessly integrated it into Unity to drive a deterministic Pong simulation via low-level DLL marshaling.

Technical Focus:

  • Cross-Language Architecture: Exposed optimized native C++ vector, matrix, and geometric operations directly to C# scripts using `DllImport` and custom P/Invoke data wrapping.
  • Memory Interoperability: Managed explicit struct layouts (`LayoutKind.Sequential`) to prevent garbage collection overhead and ensure bitwise compatibility between managed C# memory and unmanaged C++ memory.
  • Low-Level Physics Execution: Bypassed Unity's default physics engine entirely, calculating all trajectory vectors, continuous AABB collisions, and rebound logic directly inside the native library.
