NATIVE INTEROP
C++ / C# / Native Plugins

Native Math Library & Pong Clone

A technical exercise in cross-language communication, consisting of a custom 2D math library written in unmanaged C++ that runs a simple Pong simulation inside Unity via DLL plugins.

Technical Focus:

  • P/Invoke Integration: Used `DllImport` to expose raw C++ structure math functions directly to Unity's managed C# environment.
  • Memory Layouts: Applied strict `LayoutKind.Sequential` structures to ensure that data packages retained the same memory size and alignment across language barriers.
  • Physics Calculations: Calculated basic directional reflection vectors and continuous AABB collision boundaries entirely inside the C++ DLL.
