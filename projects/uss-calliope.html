Unity · C# · Gameplay

USS Calliope

8-week team prototype (5 people, 1-week sprints). I owned player combat: input, aiming, weapons, hits, hit chance / ballistics, footstep audio pool, GUID editor tool.

  • Unity
  • C#
  • Input System
  • ScriptableObjects
  • Cinemachine

What I built

  • AttackInput — Input System events (aim, fire, reload, swap weapon).
  • PlayerAimController — Mouse aim on ground plane, hold-to-aim, Cinemachine offset, crosshair color/noise from hit chance.
  • PlayerWeaponHandler — Equip weapons, semi/full auto, reload, AmmoModel + inventory.
  • PerformAttack — Hitscan, melee sphere cast, unarmed; Gaussian spread via BallisticsUtility.
  • ImpactProcessor — Damage by distance curve, melee/unarmed/stun on IDamageable.
  • SO_WeaponType — ScriptableObject weapon data (fire rate, pellets, falloff, movement penalties).
  • Hit chance — Score from range, stamina, health, movement; drives crosshair and spread.
  • Footsteps — Cadence by walk/sprint/crouch; AudioPoolManager (no per-step AudioSource alloc).
  • GUID Generator — Editor tool for ScriptableObject IDs (Tools → GUID Generator).
